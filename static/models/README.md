# 3D Models Directory

Place your 3D lamp model files here in GLB format.

## Required Files

- `lamp_sphere.glb` - Sphere lamp 3D model
- `lamp_heart.glb` - Heart lamp 3D model
- `lamp_bunny.glb` - Bunny lamp 3D model
- `lamp_cow.glb` - Cow lamp 3D model

## Converting to GLB Format

If your models are in OBJ, STL, or PLY format, convert them to GLB using:

### Option 1: Blender (Recommended)
1. Open Blender
2. File → Import → (Your format: OBJ/STL/PLY)
3. File → Export → glTF 2.0 (.glb)
4. Choose "GLB" format and export

### Option 2: Online Converter
- Use: https://products.aspose.app/3d/conversion/obj-to-glb
- Upload your file and download GLB

## Features

The GLB models will be displayed on the website with:
- Auto-rotation to showcase the design
- Mouse/touch controls for 360° viewing
- Zoom in/out capability
- Displayed alongside lamp images in the Applications section
